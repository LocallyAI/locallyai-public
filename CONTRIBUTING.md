# Contributing

This repository is a **sanitised public mirror of an active commercial project**. The canonical development happens in a private repository; this mirror exists primarily as a portfolio artifact and reference for evaluators.

## Pull requests

Pull requests are **not currently accepted** here. Changes opened against this mirror will sit unmerged because the canonical changes flow the other direction — from the private repo to this mirror.

If you've spotted a real issue (a bug, a security concern, a documentation error), please open a GitHub issue describing it. Issues *are* monitored.

## Collaboration enquiries

If you're interested in collaboration on the commercial project — engineering contribution, licensing the non-AGPL version, or a deployment partnership — please open an issue tagged `collaboration` with a short note about who you are and what you have in mind.
